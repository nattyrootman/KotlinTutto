Les fonctions minOrNull() et maxOrNull() fonctionnent avec les types de base

puisque la fonction min et max sont devenues Obsolètes selon la documentation remplacées par minOrNull() et maxOrNull() lequelles nous allons utilser dans ce tuto si vous avez une collection de types de base, vous pouvez utiliser les fonctions minOrNull() et maxOrNull() pour trouver la valeur la plus basse ou la plus élevée. Si vous voulez trouver la valeur la plus élevée dans une List< Int>, par exemple, vous pouvez utiliser le code suivant : 

 
        

fun main()  {



    val list=listOf(1,2,3,4,5)

    println(list)
  afficher liste commplete 

    val max=list.maxOrNull()
    val min=list.minOrNull()


    println("max:$max")
    println("min:$min")

      
     }
    
   

5 est le plus grand nombre dans la liste 
et 1 est le plus petit nombre dans la liste

Résultat:

        [1, 2, 3, 4, 5]
        max:5
        min:1

Explication:Les fonctions minOrNull() et maxOrNull(), cependant, ne peuvent pas être utilisées avec des types sans ordre naturel. Vous ne pouvez pas les utiliser , par exemple, avec une List<Étudiant> ou un Set<école>, car les fonctions ne savent pas automatiquement comment les Étudiants ou les objets de l'école doivent être commandés. Cela signifie que pour les types plus complexes, vous avez besoin d'une approche différente que nous verrons dans les prochains tutos
