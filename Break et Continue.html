1 Break et Continue (pause et coninuer)

Les instructions break et continue en Français(pause ou stop et coninuer ) sont les instructions jump utilisées pour ignorer certaines instructions à l’intérieur de la boucle ou mettre fin à la boucle immédiatement sans vérifier l’expression de test. Ces instructions peuvent être utilisées à l’intérieur de n’importe quelle boucle, par exemple pour, while, do-while loop. 

            

    // Simpe exemple de break dans une boucle




    fun main()  {

      for (i in 1..20){

       if (i==10)break

       println(i)
     }
    
   }

Résultat:

     1 2 3 4 5 6 7 8 9

Remarque la boucle doit commencer de 1 à 50 mais avec l' instruction break dans la condition if veut dire si i==20 alors stoppe la boucle et le reste de la boucle est ingoré....



2 break et continue dans la boucle for loop

Un autre exemple pour mieux illustrer comment l'instructions break et continue fonctionnent ici nous allons créer une liste contenant un seul élément qui est le nombre zéro pendant son initialisation et on fera rajouter des nous nous éléments dans cette liste dans la boucle avec l'instruction break et continue 


    //  exemples de break  et
     continue dans une boucle


     fun main()  {

  var list = mutableListOf(0)

  for (i in 2 until 20 step 2){

      if (i==2)continue
      if (i==10)break
          //c'est stoppé avant 10 

      list.add(i)

      println(list)
     }

  }

Résultat:

        [0, 4]
        [0, 4, 6]
        [0, 4, 6, 8]

Remarque: la boucle doit débutter de (until veut dire jusqu à ) 2 jusqu'à 20 avec un saut de 2 ça veut dire de 2 à 4 à 6 à 8 jusqu' on arrive au nombre 20 l'instruction continue veut dire ici tant que i ==2 alors continue la boucle et l instruction break veut dire si i == 10 alors il faut tout stopper mais et à la fin vous avions rajouter i dans notre liste comme élément ce qui avait donné le Résultat d'en haut.



3 Break et continue dans la boucle while(loop)

On peut également utiliser Break et continue dans la boucle while loop et do while loop comme c'etait le cas avec for loop et toutes ces difféerentes boucles vont donner les résultas identiques 

    
    fun main()  {
    
           
// initialiser avec 0 comme
// seul élément ds la liste pour le moment
   var list = mutableListOf(0)
        
    var  i=0
    while (i<20){
     i+=2
     if (i==2)continue
     if (i==10)break

      list.add(i)
//rajout de i comme élément ds la liste 
            
        println(list)
     }

  }

Résultat:

      [0, 4]            
      [0, 4, 6]      
      [0, 4, 6, 8]

Break et Continue dans la boucle dans do while

On peut également utiliser Break et coontinue dans la boucle do while loop comme c'etait le cas avec while loop et for loop et toutes ces difféerentes boucles vont donner les résultas identiques 

            
            
    fun main()  {
            
                   
// initialiser avec 0 comme
// seul élément ds la liste pour le moment
  var list = mutableListOf(0)
                
  var i=0
  do {
      i += 2
   
     if (i==2)continue
     if (i==10)break

       list.add(i)
       println(list)

     } while (i < 20)
           
        
  }

Résultat:

    
     [0, 4]             
     [0, 4, 6]       
     [0, 4, 6, 8]

Concluion:

Dans la vraie vie c'est à dire en pratique dans les vrais projets de programmation vous ne utiliserez presque jamais ou rarement ces instructions break et coninuer ou moins vous ne les rencontrez peu dans la vraie pratique ces instructions sont presqu'un heritage du monde de la programmation
